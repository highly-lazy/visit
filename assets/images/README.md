# Rasmlar / Images / Изображения

## 🇺🇿 Profil rasmini qo'yish
1. O'z rasmingizni shu papkaga ko'chiring
2. Faylni `profile.jpg` deb nomlang (yoki `profile.png` — keyin `index.html` da `src` ni o'zgartiring)
3. Tavsiya etiladigan o'lcham: **800×1066 px** (3:4 nisbat), 1MB dan kichik
4. Brauzerni yangilang — rasm avtomatik ko'rinadi

## 🇬🇧 Add profile photo
1. Copy your photo to this folder
2. Rename it to `profile.jpg` (or `profile.png` — then update `src` in `index.html`)
3. Recommended size: **800×1066 px** (3:4 ratio), under 1MB
4. Refresh the browser — the image appears automatically

## 🇷🇺 Добавить фото профиля
1. Скопируйте свою фотографию в эту папку
2. Переименуйте в `profile.jpg` (или `profile.png` — затем обновите `src` в `index.html`)
3. Рекомендуемый размер: **800×1066 px** (3:4), менее 1МБ
4. Обновите браузер — изображение появится автоматически

---

> Agar rasm bo'lmasa, sayt o'rniga chiroyli placeholder ko'rsatadi.  
> If no image is present, the site shows a styled placeholder.  
> Если фото нет, сайт покажет красивый placeholder.
